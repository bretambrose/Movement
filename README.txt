Movement

A pathfinding and movement testbed project.

Additional Installation Steps

1.  	To build or run the testing projects you will need to install NUnit.  Download from www.nunit.com
2.	You also need to install Rhino Mocks.  Download from http://builds.hibernatingrhinos.com/builds/Rhino-Mocks
	Extract the files to a top-level directory called "RhinoMocks-3.6"
